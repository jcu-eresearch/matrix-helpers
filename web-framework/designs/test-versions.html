Conditional Keywords

Conditional keywords will not function in Parse files as they only work in `Bodycopy` areas. The following test harness proves this. This code should render a button and not a failure message in Matrix 5.4, but in Matrix 5.3.x both the button and message are displayed. 

          
%begin_asset_name%
<a class="btn btn-lg btn-primary" href="%asset_url%">
I'm visible if conditional keywords succeed.
</a>
%else_asset%
<h2>This should never be visible.</h2>
%end_asset%
%begin_asset_name% I'm visible if conditional keywords succeed. %else_asset%

This should never be visible. %end_asset%
